# Listing-1.17

CSS には複数の値を同時に設定することも可能である。

```css
/* font-style: italic; */
/* font-weight: bold; */
/* font-size: 18px; */
/* line-height: 1.2; */
/* font-family: "Helvetica", "Arial", sans-serif; */
font: italic bold 18px/1.2 "Helvetica", "Arial", sans-serif;

/* background-color */
/* background-image */
/* background-size */
/* background-repeat */
/* background-position */
/* background-origin */
/* background-chip */
/* background-attachment */
background: ;

/* border-width */
/* border-style */
/* border-color */
border: ;

/* top */
/* right */
/* bottom */
/* left */
border-width: ;
```
